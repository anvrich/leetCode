# Билет 1
1)   Основные принципы JavaScript — объектно-ориентированный скриптовый язык программирования.
>Основные архитектурные черты:
> -  динамическая типизация;
>-  слабая типизация;
>-  автоматическое управление памятью;
>-  прототипное программирование;
>-  функции как объекты первого класса

2) JSP Action - это XML-элементы, позволяющие управлять поведением сервлета
>- **jsp:include** - Включает в страницу внешний файл во время обработки запроса
>- **jsp:forward** - Перенаправление на другую страницу
>- **jsp:useBean** - Добавляет на страницу экземпляр Java Bean с заданным контекстом

3) PHP делающий HTML с приветствием через GET
```
<?php
if (isset($_GET['name'])){
    echo "<h1>" . $_GET['name'] . "</h1>";
}else{
     echo <h1> . 'Name param reading error' . "</h1>";
}
?>
```
